Ember

История EmberJS: Mailroom App

Ember CLI

Интерфейс командной строки (англ. Command line interface, CLI) — разновидность текстового интерфейса (TUI) между человеком и компьютером, в котором инструкции компьютеру даются в основном путём ввода с клавиатуры текстовых строк (команд)

Должно быть:
  • nodejs
  • npm
Далее:
npm install -g ember-cli

Создание ember app

После установки Ember CLI у нас появился доступ к команде ember в нашем терминале. Команда ember new создаст нам папку с приложением внутри

ember new ember-quickstart

В новой папке с названиемember-quickstart появляется полноценное Ember приложение, включая:

  • Сервер для разработки
  • Компилятор шаблонов
  • JavaScript и CSS минификатор.
  • ES2015+ прибамбасы с помощью Babel.

Запуск ember app

Чтобы запустить приложение достоаточно выполнить одну команду внутри нашей новой папки:

cd ember-quickstart
ember server

Спустя несколько секунд мы увидим вывод подобный этому:
Livereload server on http://localhost:49152
Serving on http://localhost:4200/
Нажав Ctrl-C мы прекратим работу сервера

Angular

Настройка Angular окружения

Должно быть:

  • nodejs
  • npm

Далее:

npm install -g @angular/cli

Создание и запуск angular проекта

После установки Angular CLI у нас появился доступ к команде ng в нашем терминале. Чтобы создать проект используем команду new

ng new my-app

Запускать сервер мы будем следующими командами:

cd my-app
ng serve --open
